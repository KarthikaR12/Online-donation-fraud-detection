# Online Donation Fraud Detection System

## Project Overview
This project detects fraudulent online donation transactions using Machine Learning techniques.

## Features
- Detects fraudulent transactions
- Displays fraud type
- Real-time prediction using Flask
- Interactive frontend UI
- Trained ML classification model

## Technologies Used
- Python
- Flask
- Scikit-learn
- Pandas
- HTML, CSS
- Machine Learning

## Dataset
Simulated donation transaction dataset.

## Future Improvements
- Deploy on cloud
- Add real-time API integration
- Improve fraud classification accuracy
